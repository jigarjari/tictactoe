# Tic Tac Toe

A simple 3x3 Tic Tac Toe game made using C# Windows Forms (WinForms).

I created this project in my 4th semester of college when C# was first introduced to us as a subject. This was mainly a fun practice project to understand:
- Windows Forms
- Event handling
- Buttons and UI design
- Basic game logic in C#

## Features
- 2 Player Tic Tac Toe
- 3x3 game board
- Win and draw detection
- Simple and beginner-friendly UI

## Technologies Used
- C#
- .NET WinForms
- Visual Studio

## Purpose
This project was made for learning and fun while exploring C# and WinForms during college.
